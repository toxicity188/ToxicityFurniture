# ToxicityFurniture

## Dependency
[ToxicityLibs](https://github.com/toxicity188/ToxicityLibs)

## Add this plugin to dependency
``` kotlin
repositories {
    mavenCentral()
    maven(url = "https://jitpack.io")
}

dependencies {
    compileOnly("com.github.toxicity188:ToxicityFurniture:master-SNAPSHOT")
}
```
